# Estrutura de `Move`

Documentação da classe `Move` em `core/move.py`.

A classe `Move` encapsula os dados de um movimento de damas e é definida em
`core/move.py`.

- `start_pos`: posição inicial (linha, coluna)
- `end_pos`: posição final (linha, coluna)
- `captured_positions`: lista de posições capturadas
- `is_promotion`: booleano indicando promoção de peça

Para a documentação de API gerada, veja a seção [Core](api/core.md).
